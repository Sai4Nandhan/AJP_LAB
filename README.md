# AJP Lab – Student & Faculty API

This is a simple Project **Node.js and Express.js backend application** connected to **MongoDB** using **Mongoose**.

It provides basic REST APIs to manage **Student** and **Faculty** data.

## APIs

**Student**

* POST `/api/students` – Add a student
* GET `/api/students` – View all students

**Faculty**

* POST `/api/faculty` – Add faculty
* GET `/api/faculty` – View all faculty

## Run the Project

Install dependencies:

```
npm install
```

Start the server:

```
node Server.js
```

Server runs on **http://localhost:3000**

## Database

MongoDB database used: `studentDB`
